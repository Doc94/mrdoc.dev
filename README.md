### Prerequisites

- Node.js and npm installed.
- Git installed.

### Setup

1. **Clone the repository:**

   ```sh
   git clone https://github.com/Doc94/mrdoc.dev.git
   ```

2. **Install dependencies:**

   ```sh
   cd mrdoc.dev
   yarn install
   ```

3. **Run the development server:**

   ```sh
   yarn dev
   ```

### Usage

- **Build for production:**

  ```sh
  yarn build
  ```

- **Deploy:**

  ```sh
  yarn deploy
  ```
